# CMS Roadmap

## Future CMS Plan

Tranqill Tribe may use Sanity CMS in a future phase so non-technical members can update site content without editing code.

## Planned Content Types

- Releases
- Shows
- Press Assets
- Site Settings

## Phase Plan

- Phase 1: Static/local data remains the source of truth.
- Phase 2: Add Sanity CMS for managed content updates.
- Phase 3: May add a custom Tranqill Lab audio experience.

## Admin Portal

Do not build a custom admin portal yet.

Reason: using an established CMS reduces security risk and maintenance complexity compared with owning authentication, permissions, editing workflows, and admin infrastructure inside the app.
